Proportional Line Split- No Fields

Title Proportional Line Split- No Fields

Summary

This scripting tool will take an input feature line, and split its line features into the number of target segments specified by the target specified by the input. Each line feature will be split into the number of segments specified, each of equal length. This version of the script will not carry over any of the fields of the original feature class, it will only split the geometry into equal length segments for a number of segments for each line equal to the target count. The script will attempt to add and calculate Segment ID and Line ID fields based on field calculations using an assumed ordered ObjectID (based on how the geometry was created). IDs are based on field calculations hinging on assumptions on the object ID, so check them before using them. Beware, this script hinges on functions only available in ArcGIS 10.3. Note: This is an older script that uses copy features with geometry objects, it does not use cursors to "copy" the fields.


Usage

The goal of this script was to split target line features into (such as routes or paths) for all the line features in a feature layer, into segments of equal length similar to many of the proportional editing tools. The intended uses for this are:

  • Aid in the creation of study segments to summarize data on for networks.

  • Aid in the creation of animations for routes by allowing the creation equal length converging line segments whose ends can be converted to points.

  • Provide a tool for batch editing and segmentation of polylines.


Syntax

ProportionalLineSplitNoField (Input_Feature_Line, Target_Segment_Number, Output_Feature_Line)

Parameter Explanation Data Type
Input_Feature_Line Dialog Reference

This is the input feature class whose individual geometry features will be broken into the number of desired target segments and put into a new feature class. The input can only be a polyline for this tool, because it uses Polyline specific methods.

___________________

Python Reference

This tool depends on the segmentAlongLine method in ArcGIS 10.3.

 Feature Layer
Target_Segment_Number Dialog Reference

This input represents the target number of segments each individual row feature will be broken up into. Keep in mind these segments for this version of the tool will not keep the original feature classes fields, and thus might be a more stable version.

___________________

Python Reference

Uses a similar method for splitting up segments as specified here: https://arcpy.wordpress.com/2014/10/30/split-into-equal-length-features/

 Long
Output_Feature_Line Dialog Reference

This is the output polyline feature class that will be created by this tool. It should have object IDs, and estimated line number and segment IDs.

___________________

Python Reference

If the line numbers or segment IDs are null, then the add and calculate field elements of the tool failed.
